# Snake Pascal

## Quick Start

You can use the [Free Pascal](https://www.freepascal.org/download.html) compiler:

```console
$ fpc ./snake_game.pas
$ ./snake_game
```

## Controls

| key          | description        |
|--------------|--------------------|
| <kbd>w</kbd> | Move up            |
| <kbd>a</kbd> | Move left          |
| <kbd>s</kbd> | Move down          |
| <kbd>d</kbd> | Move right         |
| <kbd>q</kbd> | Quit               |
